========================================
Image Saver 脚本使用说明
========================================

1. 启动相机节点（在第一个终端）
   ----------------------------------------
   ros2 launch ur_bringup simulation.launch.py


2. 运行图像保存脚本（在第二个终端）
   ----------------------------------------
   
   # 默认配置：保存到 saved_images/ 文件夹，4张/秒
   python3 ~/ros2_ws/save_images.py

   # 自定义保存目录
   python3 ~/ros2_ws/save_images.py --dir my_images

   # 自定义保存频率（例如 2张/秒）
   python3 ~/ros2_ws/save_images.py --fps 2

   # 同时自定义目录和频率
   python3 ~/ros2_ws/save_images.py --dir dataset_2024 --fps 5


3. 停止保存
   ----------------------------------------
   按 Ctrl+C 停止脚本


4. 保存的图像信息
   ----------------------------------------
   - 文件名格式: image_000001_20241204_102315_123.jpg
   - 包含序列号和时间戳
   - 图像格式: JPEG


5. 查看已保存的图像
   ----------------------------------------
   ls -la saved_images/
   ls -la my_images/
